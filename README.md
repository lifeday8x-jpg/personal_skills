# personal_skills

Personal public archive for assorted Codex-style skills.

This repository is intentionally small, uneven, and personal. It is a place to keep my own skills, visual directions, prompt workflows, and workflow notes in one public location. Some skills are experimental, some are narrow, and some may only make sense for my own projects.

## Skills

### `eastern-mystic-ui-skill`

Art-direction skill for eastern mystic mobile game UI assets: Lingjian spirit-diagnostic devices, Shanhai ink landscapes, jade/brass astrolabe UI, talisman glyphs, root-bone cards, ritual buttons, and spiritual energy effects.

### `eastern-mystic-ritual-style`

Eastern fantasy ritual art system with style bible, prompt templates, layer specs, asset type definitions, consistency checks, and optional batch production workflow.

### `q-version-urban-mobile-art`

Q-version urban mobile-game art direction skill.

### `q-version-xianxia-mobile-art`

Q-version xianxia mobile-game art direction skill.

### `urhox-mobile-ui-builder`

UrhoX mobile game UI builder skill for one-click UI generation, focused UI repair, portrait/landscape layouts, AI placeholder assets, and game UI guardrails.

## Layout

Each skill lives in its own folder:

```text
.
├── eastern-mystic-ritual-style/
├── eastern-mystic-ui-skill/
├── q-version-urban-mobile-art/
├── q-version-xianxia-mobile-art/
└── urhox-mobile-ui-builder/
```

Most skill folders follow this shape:

```text
skill-name/
├── SKILL.md
├── agents/
├── references/
├── assets/
└── scripts/
```

Not every skill uses every optional folder.

## Notes

- This is not a polished marketplace package.
- Reference images are included only to communicate style direction.
- Generated ZIP packages and local runtime artifacts are ignored.
- Skills may change as I use them.
